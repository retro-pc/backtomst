                          Back to MST v1.0 (alfa)


                                1. Введение

     Программа Back to MST предназначена для работы с физическими дисками  и
образами формата CP/M - Корвет.

     Программа позволяет:
     - Формировать образы реальных дисков и записывать образы на диски.
     - Форматировать диски.
     - Копировать отдельные файлы на диск (образ) CP/M-80 и с диска (образа)
     CP/M-80.
     - Просматривать файлы  на диске  (образе диска)  CP/M-80 в виде текста,
     HEX или в виде текстового дампа.
     - Удалять отдельные файлы с дисков (образов дисков) CP/M-80.

     Программа работает на ПК под управлением операционной  системы  Windows
(была проверена работа с Windows XP). Специальная версия  под  DOS  работает
непосредственно в чистом DOS, работа в среде Windows проверялась только  при
работе с образами дисков.

     Программа еще очень сырая и  глючная,  и,  видимо,  содержит  множество
ошибок, так как в ней как таковые отсутствуют проверки  "от  дурака".  Умный
человек и сам поймет что можно, а что нельзя,  а  дураку  эта  программа  не
понадобится.    В    дальнейшем    на   http://marinovsoft.narod.ru/   будут 
выкладываться  следующие  версии   программы,  а  также  возможно  появление 
как новых программ для работы с Корветом, так и  документации  для него.

                            3. Запуск программы

     При запуске программы появится обычный для такого типа программ рабочий
стол. В верхней части экрана имеется главное меню, доступ к  которому  можно
получить клавишей F10.  Все  полезные  функции  программы  вызываются  через
главное меню. Далее последует их описание.


                               4. Меню "File"

     В этом меню имеются несколько пунктов, на данный момент из них работают
следующие:

     - "OS Shell" пункт актуален только для DOS версии программы и позволяет
временно запустить командную строку.
     - "Exit" - выход из программы.


                               5. Меню "Disk"

     Представлены следующие пункты:

     - "Read", предназначенный для чтения физического  диска  в  образ,  при
этом  запрашивается  имя  файла,  в  который  будет  осуществляться   запись
прочитанных данных.
     - "Write", позволяет записать образ диска на физический диск, при  этом
запрашивается имя файла-образа.
     - "Format", производится форматирование диска.
     - "List disk" - открывается файловая панель с каталогом диска.


                             6. Меню "Options"

     Имеет следующие пункты:
     - подменю "Drive", в котором можно  выбрать  накопитель  "А",  "B"  или
выбрать образ диска для работы.


                              7. Меню "Window"

     Позволяет  проделывать  некоторые  стандартные   операции   с   окнами:
закрывать, изменять размеры,  переключаться  между  ними  и  т.д.,  в  общем
стандартный набор операций для такого рода программ.


                               8. Меню "Help"

     Может, когда-нибудь  найдется  человек,  который  напишет  справку  для
программы, а пока там присутствует только один пункт - "About...", который в
комментариях не нуждается.
